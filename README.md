<h1> Spring boot to do app</h1>
A very simple to do application built with spring boot using H2 in memory database to store tasks

<h1> tech stack</h1>
- Java spring boot
- Spring data JPA
- H2 in memory database
- Thymeleaf 
- CSS

<h1> current featuers </h1>
- Viewll all tasks
- add new tasks with title and description
- data maintainted with H2

<h1> how to run </h1>
- Download repo
- Use an IDE like intellij
- run at "localhost:8080/"
